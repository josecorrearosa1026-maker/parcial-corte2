# Proyecto 1 - Sistema base de parqueadero

## Descripción
Este proyecto corresponde a la versión base del sistema de parqueadero.  
Su objetivo es simular la administración básica de un parqueadero, permitiendo registrar el ingreso y salida de vehículos, mostrar el mapa del parqueadero y calcular el valor a pagar según el tiempo de permanencia.

## Funcionalidades implementadas
- Generación de mapa del parqueadero con tamaño mínimo de 16x16.
- Visualización de vías, entrada, salida y espacios de parqueo.
- Registro de ingreso de carros.
- Registro de ingreso de motos.
- Registro de salida de vehículos.
- Cálculo del valor a pagar según el tiempo de permanencia.
- Organización del programa mediante funciones.
- Uso de arreglos y ciclos `for`.

## Estructura del proyecto
- `main.cpp`: contiene el menú principal y las llamadas a las funciones.
- `funciones.h`: contiene las declaraciones de las funciones.
- `funciones.cpp`: contiene la implementación de la lógica del sistema.

## Desarrollo realizado
Este proyecto fue construido como la versión obligatoria y funcional del sistema de parqueadero, cumpliendo con los requisitos mínimos del parcial. La lógica principal fue separada en funciones para mantener el programa modular y con un `main()` más limpio. 

## Innovaciones
En este proyecto no se agregaron desarrollos innovadores adicionales, ya que esta carpeta corresponde a la versión base del sistema.  
Las mejoras e innovaciones fueron implementadas en el proyecto 2.
